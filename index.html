Six changes for Version 2

Each change below shows the correct detail from the original costume next to the current detail on Version 1.

Color check: in our photos the Version-1 piping and snap caps look very dark, almost black. The original costume uses a warm, matte, medium chocolate brown for both the piping and the buttons (see photos T1–T5). Some of the difference may be our indoor lighting — please compare the actual trim against the auction photos before sewing Version 2.
4

Already correct — please keep

Version 1 got a lot right. Please keep all of these exactly the same in Version 2.

  • Piping layout: collar edge, both sides of the front placket, front and back yokes, cuff edges
  • Long pointed collar shape with piped edge
  • Rounded “shotgun” cuff shape with piping around the curved opening
  • Four closures per cuff (correct number — they just need to be buttons, not snaps)
  • Two smile pockets, correctly placed and angled on the chest
  • Plain fabric — no embroidery, matching the original
  • Shoulder width (19″) and neck size (17″)
  • Cream/off-white fabric color
