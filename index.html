Details that need improvement

Three construction changes. Green ✓ = how it must be (the original costume). Red ✗ = Version 1, which must change.

Also please check: ① the front-yoke piping curve — on the original it is a small, tight hook that hugs the shoulder seam (photo T2); on Version 1 it is a large round loop that spreads onto the sleeve. ② The piping and button color — it must be a warm, matte, medium chocolate brown, not black and not shiny (compare with the target photos).

4

Already correct — please keep

These parts of Version 1 are right. Please keep them the same.

  • Piping layout: collar, front placket, front and back yokes, cuff edges
  • Long pointed collar shape with piped edge
  • Rounded “shotgun” cuff style with piping
  • Two smile pockets, correctly placed and angled
  • Plain fabric — no embroidery
  • Shoulder width (19″) and neck size (17″)
  • Cream / off-white fabric color
